What is Wesley?

Wesley is a discord bot that scans uploaded images and generates alt text for the visually impaired.

This image contains...

How to use Wesley
(You'll need the bot installed in your discord server)
1) Find an image you wish to scan. There is a random generator here if you need.


2) Ensure Wesley has permissions to view and send messages in all server channels. This allows alt text to be generated in any given channel.


3) Upload your image to recieve auto-generated alt text and tags for your image.

Future updates:
